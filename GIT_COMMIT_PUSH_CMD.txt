Git 커밋 및 푸시 방법 (cmd에서 실행)

## 문제
- OneDrive가 파일을 잠그고 있어 PowerShell에서 Git 작업이 실패합니다
- 네트워크 연결 문제도 발생할 수 있습니다

## 해결 방법

### 1. 명령 프롬프트(cmd) 열기
- `Win + R` → `cmd` 입력 → Enter
- 또는 시작 메뉴에서 "명령 프롬프트" 검색

### 2. 프로젝트 디렉토리로 이동
```cmd
cd "C:\Users\kjaso\OneDrive\Desktop\Projects\3. Cookka"
```

### 3. Lock 파일 삭제
```cmd
del /f /q .git\index.lock
del /f /q .git\config.lock
```

### 4. Git 작업 수행
```cmd
git add .
git commit -m "Fix: Vercel build event handler error - add revalidate 0 and remove duplicate viewport meta"
git push
```

## 대안: OneDrive 일시 중지
1. OneDrive 아이콘 우클릭
2. "일시 중지" 선택
3. Git 작업 완료 후 다시 "동기화 재개"

## 네트워크 문제 해결
GitHub 연결이 실패하면:
1. 인터넷 연결 확인
2. 방화벽/프록시 설정 확인
3. 나중에 다시 시도
