PowerShell 또는 cmd에서 실행할 명령어:

1. Lock 파일 삭제:
   PowerShell:
   Remove-Item -Force -ErrorAction SilentlyContinue ".git\index.lock"
   Remove-Item -Force -ErrorAction SilentlyContinue ".git\config.lock"
   
   또는 cmd:
   del /f /q .git\index.lock
   del /f /q .git\config.lock

2. 변경사항 추가:
   git add .

3. 커밋:
   git commit -m "Fix: Vercel build timeout, viewport metadata, event handler error, and add version info"

4. 푸시:
   git push

전체 명령어 (PowerShell):
cd "C:\Users\kjaso\OneDrive\Desktop\Projects\3. Cookka"
Remove-Item -Force -ErrorAction SilentlyContinue ".git\index.lock"
Remove-Item -Force -ErrorAction SilentlyContinue ".git\config.lock"
git add .
git commit -m "Fix: Vercel build timeout, viewport metadata, event handler error, and add version info"
git push

전체 명령어 (cmd):
cd "C:\Users\kjaso\OneDrive\Desktop\Projects\3. Cookka"
