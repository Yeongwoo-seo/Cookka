# 500 Internal Server Error 해결

## 문제
`localhost:8080`에서 500 Internal Server Error 발생

## 원인
서버 사이드에서 Firebase 관련 코드가 실행되면서 오류가 발생할 수 있습니다.

## 해결 방법

### 1. 서버 로그 확인
개발 서버 터미널에서 실제 오류 메시지를 확인하세요:
- 어떤 파일에서 오류가 발생했는지
- 어떤 함수에서 오류가 발생했는지
- 정확한 오류 메시지

### 2. 코드 수정 완료
- `lib/firebase.ts`: 모든 함수에 서버 사이드 체크 추가
- `lib/firestore.ts`: 모든 함수에 서버 사이드 체크 추가

### 3. 개발 서버 재시작
변경사항을 적용하기 위해 개발 서버를 재시작하세요:

```powershell
# 개발 서버 중지 (Ctrl+C)
# 그 다음 다시 시작
npm.cmd run dev
```

또는 cmd에서:
```cmd
npm run dev
```

## 확인사항

✅ 코드 수정 완료
⏳ 개발 서버 재시작 필요
⏳ 서버 로그 확인 필요

## 다음 단계

1. 개발 서버 재시작
2. 서버 로그에서 정확한 오류 메시지 확인
3. 오류 메시지를 바탕으로 추가 수정

## 참고

만약 여전히 오류가 발생한다면, 서버 로그의 정확한 오류 메시지를 공유해 주세요.
